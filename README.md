# LoksabhaElectionCaseStudy-2019

Following is an experiement to help people who believe in meritocracy while voting, Have analyzed data declared by 141 candidates (Contesting from Chelvella, Hyderabad, Malkajgiri and Secunderabad) in their affidavits submission Election Commission. Tried to come with a linear statistical model through which we can rate these candidates by features like Age, Education, Criminal Cases against them, Income Tax Compliance, Government Dues and Normality of their earnings. 

Datasource : https://affidavit.eci.gov.in/
